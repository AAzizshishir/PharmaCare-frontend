# Next.js Frontend

## Pharma Care

### Live: https://pharmacare-frontend-omega.vercel.app

A modern frontend application built with **Next.js**, **Tailwind CSS**, **TypeScript**, and **shadcn/ui**.  
Designed for scalability, clean UI, and developer productivity.

---

## Tech Stack

- **Next.js** – React framework with SSR & SSG
- **Tailwind CSS** – Utility-first styling
- **TypeScript** – Type safety and maintainability
- **shadcn/ui** – Reusable, accessible UI components
